## 一、认证模块 (Auth)

### 1.1 登录

```
POST /auth/login
Content-Type: application/json

Request:
{
  "username": "admin",
  "password": "admin123"
}

Response data:
{
  "token": "jwt-token-string",
  "userInfo": {
    "id": 1,
    "username": "admin",
    "email": "admin@ustc.edu.cn",
    "role": "admin",
    "nickname": "管理员",
    "phone": "13800138000",
    "avatar": ""
  }
}
```

### 1.2 注册

```
POST /auth/register
Content-Type: application/json

Request:
{
  "username": "newuser",
  "password": "123456",
  "email": "newuser@ustc.edu.cn",
  "nickname": "新用户"
}

Response data:
{
  "success": true
}
```

### 1.3 获取当前用户信息

```
GET /auth/userinfo
Authorization: Bearer <token>

Response data:
{
  "id": 1,
  "username": "admin",
  "email": "admin@ustc.edu.cn",
  "role": "admin",
  "nickname": "管理员"
}
```

### 1.4 退出登录

```
POST /auth/logout
Authorization: Bearer <token>

Response data: null
```

### 1.5 修改密码

```
POST /auth/change-password
Authorization: Bearer <token>

Request:
{
  "oldPassword": "admin123",
  "newPassword": "newpass456"
}

Response data:
{
  "success": true
}
```

### 1.6 访客计数

```
GET /visitor-count

Response data: 23748   (number)
```
