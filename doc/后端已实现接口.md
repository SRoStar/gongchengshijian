## xrd.py

- POST /api/process - 处理 XRD 数据
- POST /api/upload-npy - 上传 NPY 文件
- POST /api/process-npy-group - 处理 NPY 指定数据组
- GET /api/npy-info/{filename} - 获取 NPY 文件信息

## auth.py

- POST /auth/login - 登录
- POST /auth/register - 注册
- GET /auth/userinfo - 获取当前用户信息
- POST /auth/logout - 退出登录
- POST /auth/change-password - 修改密码
- GET /visitor-count - 访客计数
